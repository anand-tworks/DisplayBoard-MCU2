Detailed Description

Module to initialize and deinitialize the ventilator

Utility functions to handle ADS1115, functions to read analog samples and convert them digital

Module to read and measure O2 % from the sensor

Module to read and measure pressure/flow rate from the pressure sensors

All sensor interfaces are available in this file.
